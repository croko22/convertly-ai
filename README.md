# Convertly AI

Convertly AI is a platform for automatically generating, editing, and previewing high-converting landing pages for your courses.

## Features

- **AI-Powered Generation:** Rapidly create landing pages using AI based on course outlines.
- **Side-by-Side Editor:** Real-time WYSIWYG editor for fine-tuning landing page copy, pricing, and features.
- **Custom Slugs:** Edit and customize the URL paths for your landing pages directly from the editor.
- **Fast Previews:** Subdomain-based routing for instant, production-like previews. 

## Getting Started

First, install dependencies:

```bash
bun install
```

Then, run the development server:

```bash
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 14 (App Router)
- React Hook Form + Zod
- Tailwind CSS
- Prisma ORM
- Vercel AI SDK
